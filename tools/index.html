Tools – Code & Markdown Viewer

Übersicht

Hier sind Viewer-Tools für die Anzeige von Code- und Markdown-Dateien aus verschiedenen Verzeichnissen des Projekts. Sie ermöglichen, Dateien relativ zum Aufrufer-Verzeichnis zu laden.

Verfügbare Tools

📄 Code Viewer

Zeigt Quellcode-Dateien mit Syntax-Highlighting und Zeilennummern an. Unterstützte Sprachen: Python, JavaScript, HTML, CSS und mehr.

Parameter:

Parameter Beschreibung Beispiel
file Dateiname oder Pfad (erforderlich) script.py
lang Sprache für Syntax-Highlighting python, javascript, html, css
title Angezeigter Titel (optional) Mein Script

📝 Markdown Viewer

Konvertiert und zeigt Markdown-Dateien (.md) mit formatiertem Text, Code-Blöcken und Syntax-Highlighting an.

Parameter:

Parameter Beschreibung Beispiel
file Dateiname oder Pfad (erforderlich) inhalt.md
title Angezeigter Titel (optional) Dokumentation

Aufrufsbeispiele:

1. Relativer Pfad (aus Unterordner):
<a href="../../tools/md-viewer.html?file=inhalt.md&title=Inhaltsübersicht">
  Dokumentation ansehen
</a>
2. Absoluter Pfad (von überall):
<a href="../../tools/md-viewer.html?file=/projekte/dobot_magician/README.md">
  Readme
</a>
Direkt öffnen Dokumentation

Wie es funktioniert

Relative Pfade: Der Viewer versucht, das Aufrufer-Verzeichnis zu ermitteln (mittels document.referrer) und löst die Datei relativ dazu auf.

Absolute Pfade: Pfade, die mit / beginnen, werden direkt geladen.

Download: Beide Viewer bieten einen „Herunterladen"-Button für die angezeigte Datei.
