Türme von Hanoi

Projekt mit dem Dobot Magician zur Lösung dieses alten Rätsels.

Die Aufgabe besteht darin, eine Anzahl von Scheiben von einem Stab auf einen anderen zu bewegen. Dabei darf immer nur eine Scheibe bewegt werden und es darf nie eine größere Scheibe auf einer kleineren liegen.
