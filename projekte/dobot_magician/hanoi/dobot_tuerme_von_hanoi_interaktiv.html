Dobot Magician: Turm von Hanoi

Eine interaktive Übung zum Anwenden des rekursiven Algorithmus. Ziel: Alle Scheiben vom Startturm A auf den Zielturm C bewegen – immer nur eine Scheibe, nie eine größere auf eine kleinere.

1. Grundidee des Algorithmus

Regeln

  1. Es darf immer nur eine Scheibe bewegt werden.
  2. Es darf nur die oberste Scheibe eines Turms genommen werden.
  3. Eine größere Scheibe darf nie auf einer kleineren liegen.

Rekursiver Lösungsplan

Um n Scheiben von A nach C zu bewegen, benutzt man B als Hilfsturm:

  1. Bewege n−1 Scheiben von A nach B.
  2. Bewege die größte Scheibe von A nach C.
  3. Bewege n−1 Scheiben von B nach C.

Für den Dobot gedacht: Ein einzelner Algorithmusschritt entspricht einem Pick-and-Place-Befehl: Scheibe ansaugen oder greifen, anheben, zum Ziel fahren, ablegen. Erst die Strategie planen, dann in Bewegungsbefehle übersetzen. 🤖

hanoi(n, start, ziel, hilfe):
    wenn n == 1:
        bewege eine Scheibe von start nach ziel
    sonst:
        hanoi(n-1, start, hilfe, ziel)
        bewege eine Scheibe von start nach ziel
        hanoi(n-1, hilfe, ziel, start)

2. Interaktive Übung

Klicke zuerst auf einen Turm mit Scheibe, dann auf den Zielturm.
