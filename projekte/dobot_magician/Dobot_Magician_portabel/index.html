Über Thonny-IDE den Dobot Magician per Python steuern

Diese Projekt beschreibt die Steuerung des Dobot Magician über die Thonny-IDE mit Python unter Verwendung der 32-Bit Version der Datei DobotDll.dll.

Ausgangspunkt war die Quelle StarterGuide-Dobot-Magician-with-Python , in der eine Lösung bereitgestellt wurde, die bei der Anwendung Fehler aufwies.

Als Ursache wurde die verwendete 64-Bit-Version von Python in Verbindung mit der 32-Bit-Version der DobotDll.dll festgestellt.

Mittels ChatGPT wurde die Lösung "Dobot_Magician_portabel" entwickelt.
Diese verlangt keine Installation von Python und DobotStudio auf dem PC.

Diese besteht aus den Dateien Dobot_Magician_portabel.zip und Python3.10.11-32bit.zip.
Beide ZIP-Archive werden heruntergeladen und entpackt. Thonny-IDE starten, unter "Ausführen - Konfiguriere Interpreter ... Lokales Python und als ausführbares Python-Programm die python.exe aus dem entpackten Ordner Python3.10.11-32bit wählen.
Dann aus Thonny heraus die Datei start.py aus dem Ordner Dobot_Magician_portabel öffnen, in der Zeile 10 COM_PORT = "COM10" anpassen und dann ausführen.
Für eine erfolgreiche Ausführung muss der Dobot Magician eingeschaltet und mit dem PC verbunden sein.

Die Datei start.py enthält die notwendigen Imports und die Startfunktion.
In ihr wird auch die Datei mein_programm.py importiert, in der die eigentliche Steuerung des Dobot Magician erfolgt.
In der Datei mein_programm.py können die eigenen Steuerungsfunktionen implementiert werden.
Die Datei mein_programm.py kann beliebig umbenannt werden, muss dann aber auch in der Datei start.py angepasst werden.
Die Anpassung der Datei start.py erfolgt, indem die Zeilen 130 bis 136 entsprechend geändert werden.

Hinweis: Da die Lösung Dobot_Python für die 64-Bit-Version von Python vorliegt, wird hier nicht weiter gearbeitet.
