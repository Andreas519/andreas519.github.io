Für die AG nutzen wir vorrangig folgende Quellen und Dokumentationen:
  • Die Programmierung des Roboterarms mittels Python ohne das sperrige Dobot Studio war eines unser Hauptsanliegen.
    Ausgehend von den Datei im Dobot Studio, wurde das Projekt Dobot_Magician_portabel erstellt.
    Inzwischen konnte das Problem der fehlenden 64-Bit-Version der DobotDll.dll gelöst werden.

    Hier ist auch ein Download mitallen notwendigen Dateien und Beispielen zui finden.
  • Dobot Python API Dokumentation
    Hier werden die von uns bisher genutzten Funktionen der Dobot Python API beschrieben. Die Dokumentation wird bei Bedarf von uns erweitert.
  • Dobot Python Hilfsfunktionen
    Um die Programmierung des Dobot Magician des Dobot Magician zu erleichtern, haben wir einige Hilfsfunktionen entwickelt.
    Diese werden in der Dokumentation beschrieben.

Die offizielle Dobot Magician API Description bietet detaillierte Beschreibungen zur Steuerung des Roboterarms.
Sie deckt Entwicklungsprozesse, Parameter für Befehle sowie Schnittstellen für verschiedene Programmiersprachen ab.

Die Dobot API Documentation beschreibt die verfügbaren Funktionen, Befehle, Parameter und Rückgabewerte zur Steuerung des Dobot Magician über eine Programmierschnittstelle.

Für den tieferen Einstieg in die Programmierung des Roboters sind die Dobot Magician Demo Description für verschiedene Programmiersprachen oder das Dobot Communication Protocol besonders hilfreich.

Leider sind diese Dokumentationen über die von Suchmaschinen angebotenen Links teilweise nicht verfügbar oder die Dateien dort schwer zu finden.
