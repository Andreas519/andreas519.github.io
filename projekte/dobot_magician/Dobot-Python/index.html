Über Thonny-IDE den Dobot Magician per Python steuern

Diese Projekt beschreibt die Steuerung des Dobot Magician über die Thonny-IDE mit Python unter Verwendung der 32-Bit Version der Datei DobotDll.dll.

Ausgangspunkt war die Quelle auf GitHub:

StarterGuide-Dobot-Magician-with-Python , in der eine Lösung bereitgestell wurde, die bei der Anwendung Fehler aufwies. Darin enthalten sind neben anderen Dateien auch die Dateien DobotDllType.py und DobotDll.dll.

Im Praxistest wurde festgestellt, dass die Lösung mit der aktuellen 64-Bit-Versionwn von Python funktioniert.

Mittels ChaTGPT wurde eine Lösung entwickelt.
Diese besteht aus den Dateien DobotDllType.py und DobotDll.dll.

  • Dobot Python API Dokumentation
    Hier werden die von uns bisher genutzten Funktionen der Dobot Python API beschrieben. Die Dokumentation wird bei Bedarf von uns erweitert.
  • Dobot Python Hilfsfunktionen
    Um die Programmierung des Dobot Magician zu erleichtern, haben wir einige Hilfsfunktionen entwickelt.
    Diese werden in der Dokumentation beschrieben.
