Über Thonny-IDE den Dobot Magician per Python steuern

Dieses Projekt beschreibt die Steuerung des Dobot Magician über die Thonny-IDE mit Python unter Verwendung der 64-Bit-Versionen der DobotDll.dll und einer aktuellen Pythoninstallation.

Ausgangspunkt war die Quelle StarterGuide-Dobot-Magician-with-Python , in der eine Lösung bereitgestellt wurde, die bei uns Probleme mit dem Zugriff DobotDll.dll verursachte.

Mittels ChatGPT wurde eine stabile Lösung entwickelt und umgesetzt.

Lade die Datei Dobot-Python.zip herunter und entpacke sie in ein Verzeichnis deiner Wahl.

Die folgende Struktur wird im Verzeichnis Dobot_Python erwartet und kann mit weiteren Projekten erweitert werden.

Lade die aktuelle Version von dobot.py herunter und speichere sie im Verzeichnis Dobot_Python.

Dobot_Python\

├── dobot.py
