74HC595 – Schieberegister Animation

Daten setzen → einschieben → latchen → Ausgänge aktualisieren

Bedienung: D = DS/SER umschalten | S = aktuelles Bit einschieben | L = Latch übernehmen | R = Reset | A = Auto | O = OE | T = Takt-Diagramm | ESC = Animation stoppen
Hinweis: Diese Webversion ist eine didaktische Simulation. Sie zeigt die Arbeitsweise des 74HC595 vereinfacht und bewusst sichtbar animiert.

Takt-Diagramm der Ein-/Ausgänge

Das Diagramm zeigt die letzten Bedienaktionen. Data ist der Wert an DS/SER. SHIFT zeigt S/SH_CP-Pulse, LATCH zeigt L/ST_CP-Pulse. QA bis QH zeigen die gespeicherten Ausgangszustände.
