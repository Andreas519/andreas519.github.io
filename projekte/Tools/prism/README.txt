Enthaltene Varianten:

zeichnen-local.html
  Funktioniert direkt per Doppelklick / file://.
  Der Programmcode ist inline eingebettet. Der Download-Link zeigt weiterhin auf dateien/zeichnen-01.py.txt.

zeichnen-server.html
  Funktioniert über einen Webserver, z. B. GitHub Pages oder lokalen Server.
  Diese Variante nutzt Prism data-src und lädt dateien/zeichnen-01.py.txt per JavaScript nach.

Hinweis:
Die Browser-Funktion "Webseite vollständig speichern" erzeugt bei Prism-Seiten oft keine saubere Projektkopie.
