# Meine Überschrift

Das ist ein Text aus einer **Markdown-Datei**.

Die Datei wird per JavaScript geladen, in HTML umgewandelt und dann in die Seite eingefügt.

## Vorteile

- Markdown-Dateien bleiben auf GitHub gut lesbar.
- Die Webseite kann die Inhalte trotzdem schön formatiert anzeigen.
- Texte lassen sich leicht austauschen oder erweitern.
