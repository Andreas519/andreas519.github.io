ESP32-WLAN-Programmstarter

Der ESP32 stellt nach dem Einschalten einen Access Point mit dem Namen „ESP32-Starter“ bereit.

  • Verbinde dein Handy, Tablet oder deinen Computer mit diesem WLAN.
  • Die auf dem ESP32 gespeicherten Programme können nun gestartet werden.

Dateien für ESP32

  • main.py
    Hauptprogramm für das Projekt, das auf dem ESP32 gespeichert werden muss.
  • programme.py
    Programm zur Steuerung der ESP32-Funktionen.
  • ADC_Messung.py
    Programm zur Messung analoger Signale.
  • Digitaloszilloskop.py
    Programm für die Anzeige digitaler Signale.
  • webseite.py
    Programm für die Erstellung der Web-Seite.
  • ESP-WLAN-AP_IF-main.py
    Ausgangsdatei für das Projekt, die auf dem ESP32 gespeichert werden muss. Sie startet die Webseite und das Programm "programme.py".
