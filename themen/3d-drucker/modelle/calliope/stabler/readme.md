# Stabler-Aufsatz fuer das Calliope Motionkit

Diese Zip-Datei umfasst Modelle zum Drucken eines Stabler-Aufsatzes fuer das Calliope Motionkit.

## Druckempfehlungen

Material: PLA oder PETG

Schichthöhe: 0.2 mm

Infill: >40% Gyroid

## Besondere Hinweise
- Bei den Teilen "Gabel_Halter" und "Servo_Halter" muss eine Druckpause eingelegt werden, um M3-Muttern einzusetzen.
- Der Ueberhang bei dem Teil "Grundplatte" muss mit Stuetzstruktur gedruckt werden.

Weitere benoetigte Normteile fuer den Bausatz:
- 4x Mutter M3 DIN 934
- 7x Sicherungsmutter M3 DIN 985
- 4x Flanschrundkopfschraube ISO 7380 M3x5
- 9x Flanschrundkopfschraube ISO 7380 M3x10
- Kugellager 
- Servos 9g
