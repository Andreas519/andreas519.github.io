# Andreas Sigismund – Young Engineers Projekte

Dieses Repository dient als öffentliche Präsentationsfläche für ausgewählte und überarbeitete Technik-, Informatik- und Physikprojekte.

Die Projekte entstehen im Umfeld der AG **Young Engineers** am **Martin-Rinckart-Gymnasium Eilenburg**.

## Zweck dieses Repositorys

Viele Projekte entstehen zunächst als Arbeitsstände, Experimente oder Vorversionen in privaten Repositories.  
Dieses öffentliche Repository enthält nur ausgewählte, überarbeitete und vorzeigbare Fassungen.

Die Veröffentlichung über GitHub soll die Materialien langfristig verfügbar machen und unabhängiger von der bisherigen Schulhomepage bereitstellen.
